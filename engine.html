My Engine
⚠️ This engine is being developed for my personal use, and it is constantly breaking changing between updates, and the features are added as I need them. I decided to open-source its code to inspire others and improve with feedbacks. Feel free to use the engine as you wish.

K3D is a small ECS game engine built in C++ along with SDL3 and OpenGL.

Why aren't using Unity, Godot or any other game engine?

I like computer graphics and low level programming, developing my own engine is being a great and fun learning experience. Also, I can combine every feature that I like in the others engines in a single one. It is lightweight, uses ECS, runs on a browser, it uses C++ and highly focused on my projects.
